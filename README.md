## sdmx-koop-provider
Transform responses from an SDMX API to a Feature Service that can be used in the ArcGIS platform.

### Getting Started
- `git clone` this repo
- `npm install`
- decide on a configuration to use. copy the key.
- start the server by typing `NODE_ENV={configKey} npm start` where `{configKey}` is what you chose from the last step
- your server is now available at `http://localhost:8080/sdmx`
- you can get details at `http://localhost:8080/sdmx/info`
